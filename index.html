Восстановление пароля на WordPress ... я потратил целый час для того чтоб восстановить пароль для "админки" WordPress. Стандартный способ UPDATE `wp_users` SET `user_pass` = MD5( 'Новый пароль' ) WHERE `user_login` = 'admin' который я обычно использую не помог, наверное это из-за того что я работал не через phpmyadmin, а через альтернативный менеджер СУБД. Так вот, через час потраченного времени, я нашел прекрасный пост о сбросе пароля на WP, вся суть в том что есть готовый хеш-код "$P$BLIwZyiB0J2XvUAsNyKQI1hyEMox0A0" для пароля "12345" именно для WP, мы просто вставляем его в поле "user_pass" и "логинимся" с паролем "12345", а логин естественно не трогаем )).
